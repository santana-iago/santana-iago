# Installation

This folder is ready to be uploaded to the public GitHub profile repository:

```text
santana-iago/santana-iago
```

## Browser method

1. Open the `santana-iago` repository on GitHub.
2. Select **Add file → Upload files**.
3. Upload the contents of this folder, preserving the `assets/` and `.github/` directories.
4. Commit the changes to `main`.

The repository root must look like this:

```text
santana-iago/
├── .github/
│   └── workflows/
│       └── snake.yml
├── assets/
│   ├── wave-top.svg
│   ├── wave-bottom.svg
│   └── cards/
├── README.md
└── SETUP.md
```

## Enable the contribution animation

1. Open **Settings → Actions → General**.
2. Under **Workflow permissions**, select **Read and write permissions**.
3. Save the setting.
4. Open **Actions → Generate contribution snake → Run workflow**.
